FILE FORMAT GUIDE:
==================

.prp files (XML - Human/Machine Readable):
   - Program metadata
   - File IDs and names
   - Can open in text editor
   - Example: BGStart.exe.prp

.gbf files (Ghidra Binary Format):
   - Contains decompiled code
   - Function definitions
   - Cross-references
   - Requires Ghidra or parser

program_index.json (JSON - Machine Readable):
   {
     "name": "BGStart.exe",
     "type": "Program",
     "file_id": "c0a812a7c78e2551164125100"
   }
